Class AntColony

The Ant Colony is the objective the player must defend. It is divided into a set of tunnels, each divided into sections. Only one Ant can occupy each section at a time, except for Guard Ants. Some tunnel sections may be filled with water, which is impassable to most Ants. The Ant Colony has a stock of Food which is used to deploy Ants, as well as Boosts which have special effects when given to Ants.
